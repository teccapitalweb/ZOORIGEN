# ZOORIGEN web

Sitio estático listo para GitHub Pages.

## Estructura
- `index.html`
- `styles.css`
- `script.js`
- `data.js`
- `assets/img/...`

## Publicación en GitHub Pages
1. Sube todo el contenido del proyecto a un repositorio.
2. En GitHub entra a **Settings > Pages**.
3. En **Source** selecciona la rama principal y carpeta `/root`.
4. Guarda y usa el enlace que genere GitHub.
